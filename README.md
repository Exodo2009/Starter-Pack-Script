# Starter Pack Script

This script made for arch linux is commited to install all posibly needed files.

I will made it compatible with other distros in the future.

## System Compatibility:

- Arch Linux

    - Endeavour OS

    - Gurada OS

## Desktop Environment (DE) Compatibility:

- KDE

- Hyprland

- Qtile

- Gnome

- Xcfe
